# ⚡ 3-MINUTE QUICK START

## ❶ GET API KEY (1 minute)

1. Go to: https://console.groq.com
2. Sign up or log in (FREE)
3. Go to: API Keys section
4. Click: "Create New API Key"
5. Copy the key (starts with `gsk_`)

```
Example key:
gsk_w1X2Y3Z4a5B6c7D8e9F0g1H2i3J4k
```

## ❷ UPDATE .env FILE (1 minute)

**Open file:** `c:\Users\tvipi\project\Internship\Deccan ai\.env`

**Find this:**
```
GROQ_API_KEY=gsk_placeholder_your_api_key_here
```

**Replace with:**
```
GROQ_API_KEY=gsk_w1X2Y3Z4a5B6c7D8e9F0g1H2i3J4k
```

**Save:** Ctrl+S

## ❓ Something Wrong?

**Error: "Groq API key not configured!"**
→ Edit .env, check you pasted the full key, save, restart

**Questions are slow**
→ Normal! First question takes 3-5 seconds.

**Same question twice?**
→ Expected. Different wording, same skill topic.

**App crashes?**
→ Check internet connection. Groq needs to reach cloud API.
